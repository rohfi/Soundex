Langkah-langkah menggunakan/mengoperasikan Aplikasi Sondex berbasis Desktop

1.Buka folder Soundex_MenujuSukses
2.Masuk kedalam folder Dist.
3.Buka file Soundex_MenujuSukses.jar

Cara Penggunaan aplikasi soundex
1. Buka Aplikasi Soundex_MenujuSukses.jar
2. Setelah Aplikasi berhasil di run, maka klik "Mulai".
3. Masukan huruf di kedua String tersebut.
4. Setelah memasukkan huruf tersebut, klik "Periksa".
5. Akan keluar hasil Soundex dari kedua String tersebut, Beserta kesimpulannya.
6. Terdapat fitur info yang dilambangkan dengan huruf i untuk mengetahui apa itu Soundex.
7. Terdapat fitur kembali untuk kembali ke page sebelumnya.
8. Terdapat fitur keluar untuk keluar dari Aplikasi Soundex tersebut.
